# RPG Skills Addon

A comprehensive RPG skills system for Minecraft Bedrock Edition.

## Features

- **9 Skills**: Mining, Woodcutting, Farming, Combat, Archery, Fishing, Alchemy, Agility, Defense
- **Quests System**: Daily quests and active quests
- **Economy**: Coins system with shop
- **UI**: Clean, user-friendly interface
- **Localization**: Support for English (en_US) and Vietnamese (vi_VN)
- **Anti-Exploit Measures**: Block tracking to prevent farming

## Installation

1. Download the behavior pack and resource pack
2. Import them into your Minecraft world
3. Enable both packs

## Usage

- **Menu Compass**: Open the main menu
- **Skill Tome**: Open skills menu or cast last selected ability (sneak + use)
- **Quest Journal**: View active quests
- **Coin Pouch**: Check your balance
- **Crafting Guide**: View crafting recipes
- **Admin Wand**: Access admin dashboard (requires rpg_admin tag)

## Skills

- **Mining**: Gain XP by breaking ores and deepslate
- **Woodcutting**: Gain XP by chopping logs
- **Farming**: Gain XP by harvesting crops
- **Combat**: Gain XP by killing mobs with melee attacks
- **Archery**: Gain XP by killing mobs with projectiles
- **Fishing**: Gain XP by catching fish
- **Alchemy**: Gain XP by brewing potions and enchanting items
- **Agility**: Gain XP by running and jumping
- **Defense**: Gain XP by taking damage

## Quests

Complete daily quests to earn coins and XP. Quests refresh daily.

## Shop

Purchase items with coins earned from quests and activities.

## Configuration

The addon can be configured by modifying the `config.js` file in the behavior pack.
